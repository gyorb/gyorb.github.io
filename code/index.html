CodeChecker

With CodeChecker we created a tool to view and manage the results found by multiple static analyzers like Clang Static Analyzer and Clang Tidy.
You can check your C/C++ project easily and filter/view the results on a web interface or in the command line. With the command line interface integration to the Continous Integration loops can be done easily.
